# CLAUDE.md

This project has an `AGENTS.md` file with comprehensive guidelines for AI agents.

**See [AGENTS.md](./AGENTS.md)** for:

- Code style guidelines (imports, naming, error handling, testing)

Quick reference:
- `make test` - Run all tests
- `make lint` - Lint via golangci-lint
- `make build` - Build the binary
- Always use `rg` (ripgrep) instead of `grep`
